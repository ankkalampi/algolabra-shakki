# Yksinkertainen shakkibotti

## Asennus

1. Lataa ja käynnistä HY:n AI-platform näiden ohjeiden mukaisesti:
https://algolabra-hy.github.io/aiplatform-fi

2. Kloonaa tämä repositorio haluamaasi sijaintiin näin:
```
git clone git@github.com:ankkalampi/algolabra-shakki.git <haluamasi sijainti>
```

3. Käynnistä AI-platform, valitse vasemmasta yläreunasta "chess", ja anna kansioksi kohdassa 1. kloonaamasi projektin juurikansio, ja paina "submit".

4. Tee ensimmäinen siirto valkoisella, ja peli alkaa.
